## 🎮 Quick Actions

### Create a Role:
1. Go to http://localhost:5000/manage
2. Select your server
3. Click "Roles" tab
4. Click "Create Role"
5. Enter name and color
6. Done! Role is in Discord!

### Add a Badge:
1. Go to "Badges" tab
2. Click a badge type (Admin, Moderator, etc.)
3. Badge role is created
4. Assign to members!

### Decorate Your Server:
1. Go to "Decoration" tab
2. Click "Change Icon" or "Change Banner"
3. Enter image URL
4. Server is decorated!

### Create a Channel:
1. Go to "Channels" tab
2. Click "Create Channel"
3. Enter name and type
4. Channel appears in Discord!
