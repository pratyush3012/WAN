## 🏅 NEW: Badge System

### What It Does:
Automatically assigns visual badges to members based on their roles and permissions, making it easy to identify staff, members, and guests.

### Available Badges:
- 👑 **Owner** - Server owner
- ⚡ **Administrator** - Full admin permissions
- 🛡️ **Manager** - Server management
- 🔨 **Moderator** - Moderation permissions
- 💚 **Helper** - Message management
- ✅ **Member** - Verified members
- ⭐ **VIP** - VIP/Premium members
- 💎 **Booster** - Server boosters
- ✓ **Verified** - Verified members
- 👤 **Guest** - No badge (no roles)

### Commands:
```
/badge                  - View your badge
/badge @user            - View someone's badge
/badges                 - View all available badges
/badge-stats            - View badge distribution (Admin)
/assign-badge-role      - Create visual badge roles (Admin)
/auto-assign-badges     - Auto-assign badges to all (Admin)
```

### Quick Setup:
```
1. /assign-badge-role     → Creates badge roles
2. /auto-assign-badges    → Assigns to all members
3. Done! Members now have visual badges
```
