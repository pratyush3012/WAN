## 🚀 Quick Start (2 minutes)

### 1. Start the Bot
```bash
python3 bot.py
```

### 2. Link Your Roblox Account
In Discord:
```
/roblox-link YourRobloxUsername
```

### 3. View Your Stats
```
/roblox-stats
```
You'll see realistic demo stats like:
- Playtime: 45h 23m
- Coins: 45,230
- Kills: 234
- Level: 28

### 4. Open Web Dashboard
```
/web
```
Explore the full web interface!

## 🎯 Available Commands

```
/roblox-link <username>       - Link your Roblox account
/roblox-stats [@user]         - View player stats
/roblox-leaderboard <type>    - View leaderboards
/clan-stats                   - View clan overview (Admin)
/roblox-sync-bloxlink         - Auto-link all members (Admin)
/web                          - Open web dashboard
```
