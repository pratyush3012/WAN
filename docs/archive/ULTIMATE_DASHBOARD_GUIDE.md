## 🚀 Quick Start

### From Discord

Simply type in any channel:
```
/web
```

The bot will:
1. Generate a secure access token
2. Create a personalized dashboard link
3. Send you an ephemeral message (only you can see it)
4. Open the dashboard in your default browser

### Access Levels

Your dashboard access is automatically determined by your Discord role:

| Discord Permission | Dashboard Role | Access Level |
|-------------------|----------------|--------------|
| Administrator | Admin | Full Control |
| Manage Server | Manager | Server Management |
| Moderate Members | Moderator | Moderation Tools |
| Manage Messages | Helper | Support Tools |
| None | Member | View Only |

## 🔐 Security Features

### Token-Based Authentication

**How it works:**
1. User runs `/web` command in Discord
2. Bot generates secure token (32 bytes)
3. Token includes user ID, guild ID, and role
4. Token expires after 24 hours
5. User clicks link to authenticate
6. Session created with encrypted data

**Security Measures:**
- Tokens are single-use
- Encrypted transmission
- Secure session cookies
- HTTP-only cookies
- SameSite protection
- CSRF protection

## 🔧 Configuration

### Environment Variables

```env
# Dashboard URL
DASHBOARD_URL=http://localhost:5000

# Dashboard Settings
ENABLE_DASHBOARD=true
DASHBOARD_HOST=0.0.0.0
DASHBOARD_PORT=5000
DASHBOARD_SECRET_KEY=your_secret_key_here

# Security
SESSION_COOKIE_SECURE=True
SESSION_COOKIE_HTTPONLY=True
SESSION_COOKIE_SAMESITE=Lax
```

### Bot Configuration

In Discord, use admin commands:
```
/web-admin          # Open admin dashboard
/web-status         # Check dashboard status
```
