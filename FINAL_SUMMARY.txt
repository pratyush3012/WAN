INSTALLATION (10 MINUTES)
================================================================================

1. Clone Repository
   git clone <repo-url>
   cd wan-bot

2. Install Dependencies
   pip install -r requirements.txt

3. Create Directories
   mkdir -p ./data/watch_party
   mkdir -p ./data/music
   mkdir -p ./uploads/watch_party
   mkdir -p ./logs

4. Setup Environment
   Create .env file with:
   - DISCORD_TOKEN
   - DISCORD_GUILD_ID
   - FLASK_HOST/PORT
   - Feature flags

5. Create bot.py
   (See INSTALLATION.md for code)

6. Create app.py
   (See INSTALLATION.md for code)

7. Start Services
   Terminal 1: python bot.py
   Terminal 2: python app.py

8. Verify
   - Bot online in Discord
   - API running on localhost:5000
   - Test endpoints
