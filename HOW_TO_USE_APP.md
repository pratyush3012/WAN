# 🚀 How to Use WAN Bot.app

## ✅ The App is Ready!

You should now see **"WAN Bot.app"** in your Finder window.

## 📍 Current Location

The app is in your current directory:
```
WAN bot/WAN Bot.app
```

## 🎯 How to Use

### Option 1: Use From Current Location
1. **Double-click** `WAN Bot.app` in Finder
2. If macOS asks, click **"Open"** to allow
3. Terminal will open and bot will start
4. Web dashboard will open automatically

### Option 2: Move to Applications (Recommended)
1. **Drag** `WAN Bot.app` to your **Applications** folder
2. Open **Applications** folder
3. **Double-click** `WAN Bot` to start
4. Or use **Spotlight**: Press `⌘ + Space`, type "WAN Bot"

## 🔒 First Time Opening

macOS may show a security warning because the app is not from the App Store.

**To open:**
1. **Right-click** (or Control+click) on `WAN Bot.app`
2. Select **"Open"**
3. Click **"Open"** in the dialog
4. This only needs to be done once!

## 🌐 What Happens

When you open the app:

1. ✅ **Terminal opens** - Shows bot logs
2. ✅ **Bot starts** - Connects to Discord
3. ✅ **Dashboard opens** - Web interface at http://localhost:5000
4. ✅ **Ready!** - Bot is serving your Discord server

## 📊 Management Dashboard

The web dashboard opens automatically with:
- Real-time bot status
- Server management
- Roblox stats & leaderboards
- Badge system
- Economy features
- Moderation tools
- And more!

## 🎮 Discord Commands

Once running, use in Discord:
```
/badge              - View your badge
/roblox-link Test   - Link Roblox account
/web                - Open dashboard
/help               - See all commands
```

## 🛑 To Stop the Bot

1. Go to the Terminal window
2. Press `Ctrl + C`
3. Or just close the Terminal window

## 🔄 To Restart

Just double-click `WAN Bot.app` again!

## 💡 Pro Tips

1. **Add to Dock**: Drag app to Dock for quick access
2. **Bookmark Dashboard**: Save http://localhost:5000 in browser
3. **Use Spotlight**: Fastest way to open (⌘+Space → "WAN Bot")
4. **Keep Terminal Open**: See real-time logs

## 🐛 Troubleshooting

### Can't see the app?
```bash
# Open Finder to the app location
open -R "WAN Bot.app"
```

### Security warning?
- Right-click → Open (instead of double-click)
- Or: System Preferences → Security & Privacy → Allow

### App won't start?
- Check `.env` file has Discord token
- Make sure Python 3 is installed
- Check Terminal for error messages
