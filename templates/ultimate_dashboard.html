Translation Settings

React 🌐 on any message to instantly translate it to English. Members can then pick any language.

How it works:

  1. Someone sends a message in any language
  2. Any member reacts with 🌐 on that message
  3. Bot instantly replies with English translation
  4. Language picker buttons appear for other languages
