Privacy Policy

Last updated: March 19, 2026

This Privacy Policy explains what data WAN Bot collects, how it is used, and your rights regarding that data. By using WAN Bot, you agree to this policy.

1. Data We Collect

WAN Bot collects only the data required to provide its features:

  • Discord IDs — User IDs, server (guild) IDs, channel IDs, and role IDs
  • Activity data — Message counts, voice time, XP/level progress, economy balances
  • Configuration data — Server settings, automod rules, welcome messages, ticket configs
  • Moderation data — Warnings, bans, kicks, and mod log entries
  • Roblox data — Roblox usernames and IDs voluntarily linked by users via /roblox-link
  • Dashboard sessions — Temporary session tokens for web dashboard authentication

2. What We Do NOT Collect

  • Message content (beyond what automod needs to function, which is never stored)
  • Passwords or payment information
  • Personal information such as real names, emails, or addresses
  • Data from servers the Bot is not in

3. How We Use Data

Collected data is used exclusively to:

  • Provide Bot features (leveling, economy, moderation, tickets, etc.)
  • Display server statistics on the web dashboard
  • Enforce server-configured automod and anti-raid rules
  • Authenticate dashboard sessions

We do not sell, share, or use your data for advertising or any commercial purpose.

4. Data Storage

Data is stored in a local SQLite database on the server hosting the Bot. Dashboard session data is stored in memory and expires after 24 hours. No data is sent to third-party analytics services.

5. Data Retention

Data is retained as long as the Bot remains in your server. When the Bot is removed from a server, server-specific configuration data may be deleted upon request. User activity data (XP, economy) persists until manually cleared by a server administrator.

6. Your Rights

You have the right to:

  • Request deletion of your personal data by contacting the bot owner
  • Remove the Bot from your server at any time, stopping further data collection
  • Unlink your Roblox account using /roblox-unlink

7. Third-Party Services

WAN Bot may interact with:

  • Discord API — Subject to Discord's Privacy Policy
  • Roblox API — Only when users voluntarily link their Roblox account, subject to Roblox's Privacy Policy
  • Google Translate (free tier) — Used for the translation feature; no user data is stored by this service

8. Children's Privacy

WAN Bot does not knowingly collect data from users under 13. If you believe a child under 13 has provided data, please contact us for removal.

9. Changes to This Policy

We may update this policy at any time. Continued use of the Bot after changes constitutes acceptance of the updated policy.

10. Contact

For privacy-related requests or questions, contact the bot owner via Discord or join the support server.
